# 💰 Expense Tracker Project

A simple and beginner-friendly **Python Command Line Expense Tracker** developed as part of my **Decode Labs Internship**. This project helps users track daily expenses and calculates total and average spending.

---

## 🚀 Project Overview

This application allows users to enter multiple expenses and then displays:

* List of all expenses
* Total amount spent
* Average expense

It is built using basic Python concepts like loops, lists, conditionals, and arithmetic operations.

---

## ✨ Features

✅ Add multiple expenses
✅ Stop input anytime by typing `done`
✅ Displays all recorded expenses
✅ Calculates total spending
✅ Calculates average expense
✅ Simple and interactive CLI program

---

## 🛠️ Technologies Used

* Python 3.x
* Command Line Interface (CLI)

## ▶️ How to Run the Project

### Requirements

* Python 3 installed on your system

### Steps

1. Download or clone the project files
2. Open terminal or command prompt in the project folder
3. Run the program using:

```bash
python expense_tracker.py
```

4. Enter expenses one by one
5. Type `done` to finish input
